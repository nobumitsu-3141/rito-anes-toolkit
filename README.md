# 小児麻酔オールインワン｜離島麻酔ツールキット

体重・年齢から、小児麻酔に必要な数値を一括算出するオフライン対応の意思決定支援ツール（PWA）。

- **公開URL**: https://nobumitsu-3141.github.io/rito-anes-toolkit/
- **構成**: 単一ファイルに近い静的Webアプリ（外部依存ゼロ）＋ Service Worker による完全オフライン
- **タブ**: 気道 / 薬剤 / 緊急(PALS) / 区域・持続 / 輸液・輸血 / バイタル
- **機能**: 年齢→体重推定、主要値のコピー、全タブの印刷/PDF、ホーム画面追加でオフライン起動

## 使い方（オフライン）
1. iPhone/Mac で上記URLを一度開く
2. 「ホーム画面に追加」（iOS Safari）／インストール（Chrome）
3. 以後は電波がなくても起動・動作

## 更新方法
`index.html` 等を変更したら、`sw.js` の `CACHE` バージョン文字列（例 `rito-peds-v2.0.0`）を上げてからpush。古いキャッシュは自動削除され、次回起動で新版に更新される。
